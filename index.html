Welcome to PictureShare.

PictureShare is supposed to be a simple demo application for Windows with the posibility to upload pictures from an external HDD. Main purpose of this application is showcasing the possibilities of C# programming and is not ready for production and therefore without any warranty.

PictureShare is thought to be an application that can be easily extended for uploading pictures to other places than local HDD (see source folder MenuModules). Even the surface is a module, therefore you can i.e. replace it with WPF. TestConsole shows examples of how to use it.

PictureShare recognizes attached USB devices and stores information about them in registry. After that it pops up a menu for the user to choose where to upload the pictures to. An optional album name can be provided.

License respected you can use it for what ever you like.
